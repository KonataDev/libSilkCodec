## libSilkCodec
Cross-platform silk codec wrap library dependents on [silk-v3-decoder](https://github.com/kn007/silk-v3-decoder).

## Build
```bash
  # just make it
  $ cmake . && make -j8

  # or
  $ chmod +x build.sh
  $ ./build.sh
```
